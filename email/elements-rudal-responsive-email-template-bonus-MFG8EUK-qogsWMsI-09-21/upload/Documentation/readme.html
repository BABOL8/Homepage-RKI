Ready-to-use version

The ready-to-use (Plain HTML) inline version only requires a few updates:

  1. Images need to be hosted online
  2. Paths need to be updated accordingly Example is as below: 
    <a href="#"><img src=".../images/logo.png" width="8" height="8"/></a>

    Change it to:

    <a href="yourdomain"><img src="yourserver/images/logo.png" width="8" height="8"/></a>
    Run a quick test once you have finished editing the code.

Stampready

StampReady is a new and improved email platform where users can Create & Send campaigns simplified. We believe this process requires 3 important steps: Pick, Edit and Send. All other unnecessary features have been excluded to minimize the costs and increase the user experience.

  1. Login to Stampready dashbor.
  2. (Left Click) New Campaign - Import Template - Zip file. For reference see screenshot:
  3. Select Stampready folder and choose Stampready.zip
  4. Use template - Give your campaign a name - Start editing
  5. Now ready to use the template. Drag & Drop the thumbnail - Style it - Template Ready - Send Campaign.

Mailchimp

MailChimp helps you email the right people at the right time. Send automated emails based on customer behavior and preferences. If you are not aware to mailchimp then below is the process to import email template in your mailchimp account.

  1. Login to mailchimp website using your username & password.
  2. Click Templates - Create Template - Import Zip file.
  3. Give template name - Select Mailchimp folder - Choose Mailchimp.zip - Upload.
  4. Loading - Start editing - Save
  5. Select drop menu (edit button) - Create Campaign
  6. Follow step - Choose module - Send Campaign

Campaign Monitor

Campaign Monitor is an online email marketing application that enables designers to create, send, manage and track branded emails for themselves and their clients with ease. If you are not aware to Campaign Monitor then below is the process to import email template in your Campaign Monitor account.

  1. Login to Campaign Monitor website using your username & password.
  2. Click Templates - Import.
  3. Give template name - Select Campaign Monitor folder - Choose index.html & images.zip - Upload.
  4. Importing and scanning - Select Campaign - Create Campaign
  5. Follow step
  6. Choose template - Choose module - Send Campaign
